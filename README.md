### рҹ”§ [gogenetic](https://github.com/athishsreeram/gogenetic) вҖ” Go Microservice Ecosystem Generator

> **126 commits В· v1.0 released В· вӯҗ Most starred project**

A production code generation engine that scaffolds complete Go microservices from a TOML business entity definition. Feed it your domain model вҖ” get back a fully-wired, deployable service.

**What it generates:**

- Full project structure with routing, DB, and domain layers pre-configured
- 8 architectural patterns out of the box

```
CRUD              вҶ’ archType=crud | archType=crud-object
Event Sourcing    вҶ’ es-sync-with-cqrs | es-async-with-cqrs | es-cqrs-eventstore-read
Event Handlers    вҶ’ es-async-cmdhandler-with-cqrs | es-async-eventhandler-with-cqrs
File Output       вҶ’ archType=file  (custom template injection)
```

**Config-driven architecture switching вҖ” no code rewrite needed:**

```toml
[Gogenetic]
ArchType     = "es-async-with-cqrs"
Lang         = "go"          # go | java
TomlDir      = "./toml/todo.toml"
OutDir       = "output"
```

`Go` `Smarty Templates` `TOML` `T-SQL` `gRPC` `CQRS` `Event Sourcing` `Code Generation`
